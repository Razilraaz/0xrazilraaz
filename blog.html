Subdomain Finder

Building a Simple Subdomain Finder Script: A Step-by-Step Guide For my GitHub project, I recently developed a simple yet effective subdomain finder script using Bash. This script automates the discovery of subdomains for a given domain and checks which ones are live. Below, I provide a detailed breakdown of how the script works, its purpose, and how you can use it in your cybersecurity or penetration testing tasks. Purpose Subdomain enumeration is an essential step in reconnaissance during penetration testing. Subdomains often house lesser-known services and applications that could be vulnerable. This script simplifies the process by combining two powerful tools: assetfinder and httprobe. Prerequisites Before running the script, ensure you have the following installed on your Linux system: Bash Shell: Most Linux distributions have it pre-installed. assetfinder: A tool to find subdomains related to a domain. httprobe: A utility to check if the discovered subdomains are live. The script automatically installs these tools if they are not already present.
